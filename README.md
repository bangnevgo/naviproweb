# NaviProWeb

NaviProWeb is a web platform for managing AI-powered skills and automation workflows.

## Features

- Skill management
- Pipeline automation
- AI-powered content generation
- Modular architecture

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
